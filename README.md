## 🐍 Security Automation with Python
Leveraging my background in software engineering, I develop custom Python scripts to automate threat detection, incident response, and vulnerability management.

### 🛠️ Automation Use Cases
* **Log Analysis & Threat Hunting:** Created scripts to parse large `.log` and `.json` datasets, using regex and logic to flag IOCs (Indicators of Compromise) such as suspicious IP geolocations or multiple failed SSH login attempts.
* **Automated OSINT:** Developed tools to query APIs (like VirusTotal or Shodan) to automatically vet suspicious URLs or IP addresses found in network traffic.
* **Network Scanning & Auditing:** Built custom wrappers for `Nmap` to schedule periodic scans of local infrastructure and alert on newly opened ports or unauthorized services.
* **API Security Testing:** Scripts to automate "fuzzing" and credential stuffing tests on REST APIs to ensure robust error handling and rate-limiting.

> **Why Automation?** By automating repetitive manual tasks, I reduce the Mean Time to Detect (MTTD) and allow security teams to focus on high-level strategic defense.
